
 
 ## Running locally
 1. Install python 3.7.4
 
 1. Set up requirements. 
    
    ```bash
    pip install -r tests/requirements.txt
    ```
    
 1. Verify set up
    
    ```bash
    export PYTHONPATH=./src
    pytest
    ```
